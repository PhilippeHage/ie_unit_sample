# IE NLP Utils

Works on Python 3 under any operative system.

To install locally
(requires setuptools >= 40.9.0, released Apr 3 2019):

```
$ pip install .
```

TODO: What about `--editable` installs?
See [this thread](https://github.com/pypa/setuptools/pull/1675#issuecomment-479788797)

To generate source distribution and wheels
(requires pep517 >= 0.5.0):

```
$ python -m pep517.build --source --binary --out-dir dist/ .
```
